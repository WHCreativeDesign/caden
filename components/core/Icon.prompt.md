Line icon in Caden's house style — 24px grid, 1.7 stroke, round caps; use for any glyph in the system instead of an icon font or emoji.

```jsx
<Icon name="mic" size={22} />
```

Available: mic, send, chat, activity, shield, device, wrench, user, clock, bell, plus, arrowRight, sparkle, sun, home, x, pause, check. Inherits `currentColor`, so set the parent's `color`. Need one that isn't here? Drop in the matching Lucide path — same grid and stroke.
