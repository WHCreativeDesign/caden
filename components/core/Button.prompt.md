Caden's action button — mono label, soft corners, warm terracotta fill; use for any primary or secondary action.

```jsx
<Button variant="primary" iconRight="arrowRight">Open Caden</Button>
```

Variants: `primary` (terracotta, white-warm text), `ghost` (paper + warm border), `quiet` (text only), `ink` (for dark espresso panels). Sizes `sm | md | lg`. Pass `icon` / `iconRight` (any `Icon` name) and `block` for full width. Press settles down; hover handled in the directory card's CSS.
